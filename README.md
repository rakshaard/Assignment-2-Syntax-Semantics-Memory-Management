# Assignment-2-Syntax-Semantics-Memory-Management
Comparative analysis of syntax, semantics, and memory management using Python, JavaScript, C++, Java, and Rust.

This repository contains the source code, analysis programs, screenshots, and report for Assignment 2 on Syntax, Semantics, and Memory Management. The assignment focuses on comparing different programming languages based on syntax handling, semantic behavior, and memory management techniques.

The project includes implementations and analysis using Python, JavaScript, C++, Java, and Rust.

Objectives

The main objectives of this assignment are:

Analyze syntax errors in different programming languages
Compare how compilers and interpreters report errors
Study semantic features such as type systems and type coercion
Explore memory management approaches in Rust, Java, and C++
Understand performance, safety, and memory handling differences among languages
Practice version control using GitHub
